# 🔐 Password Security Tools

A collection of Python-based tools for testing and improving password security. Designed for learning, ethical hacking practice, and demonstration of practical cybersecurity skills.

---

## 🛠 Tools Included

1. **Password Strength Checker**
   - Checks password strength based on length, uppercase, lowercase, numbers, and special characters.
   - Outputs: Weak ❌, Moderate ⚠️, Strong ✅
   - Example Usage:
     ```bash
     python3 password_checker.py
     ```

2. **Brute Force Simulator (Dictionary Attack)**
   - Simulates a dictionary attack against a target password.
   - Tracks attempts and time taken.
   - Supports custom wordlists.
   - Example Usage:
     ```bash
     python3 bruteforce_simulator_advanced.py
     ```
